Download the MSI files (or extract from the ZIP File)
Install the application. 
If required, if the Windows Desktop runtime is not installed (windowsdesktop-runtime-9.0.16-win-(x64/x86/arm64) it will be requested during the 1st run of the application
Once installed, follow the instructions located under the 'help' menu to enable the 'LeMansUltimateTelemetryPlugin.dll' to receive telemetry from LMU.

Start LMU and go in to practice session with the car you want to setup. 
Check that the service is connected, and that the team name is showing. 
Then, select the specific car, set the RPM settings, delay and volume you require and click on save.
When Loading a the same car with a different team, you will need to select the car to associate it, once selected, the saved setting will be loaded. 
Click Save to add the new association.
